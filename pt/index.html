Proteção de ID

Esta página permite-lhe ocultar facilmente os dados privados do seu documento de identidade. Por vezes podem pedir-lhe uma cópia do documento, mas o que não deve fazer é enviar uma foto tal como está, pois se cair em mãos erradas pode causar-lhe graves problemas.

Aqui pode bloquear os dados que não são necessários e sobrescrever um texto para que não o possam utilizar para outros fins, desta forma a imagem que partilhar será única para evitar que a reutilizem.

Para começar, clique no botão de escolher foto e continue com os passos seguintes.
Todo este processo é realizado apenas no seu navegador, não são enviados dados para lado nenhum.

Perguntas e respostas

O que faz a página?

Trata-se de seguir os passos que a polícia recomenda para proteger o seu documento de identidade, sem necessidade de se lembrar de todos e saber que dados são importantes.

  1. Escolha uma foto do seu documento. A imagem é convertida para preto e branco e o seu tamanho é limitado para não dar uma cópia de alta qualidade.
  2. Escolha o formato de documento que tem e se é a parte da frente ou de trás. Com base nisso, ocultam-se os dados que não são necessários que ninguém veja.
  3. Ajuste se necessário a posição da foto para que se veja bem na pré-visualização.
  4. Adicione um texto que será sobrescrito como marca de água, indicando para quê partilha a foto.
  5. Clique no botão para descarregar a foto.
Porquê usá-la?

Partilhar o seu documento de identidade tem o risco de acabar em mãos erradas e sofrer uma usurpação de identidade. De modo geral, as empresas não deveriam ter acesso à maioria dos dados do seu documento. Pode ler este pdf com o relatório 48/2023 da AEPD. Já houve casos de sanções a empresas, pelo que se enviar esse link à empresa que lhe pede o documento talvez pensem duas vezes.
E claro, a nível particular não faz sentido enviar uma cópia ou foto do seu documento a outra pessoa. Desconfie se alguém lhe pedir um documento para uma compra-venda!

Se tem a certeza de que precisa enviar-lhes o documento, o que deve fazer é seguir as diretrizes da Polícia Nacional , de forma que essa cópia esteja a preto e branco, oculte os dados que não são necessários para o identificar e seja sobrescrito um texto para que o uso seja único.
É precisamente isso que esta página faz de forma automatizada.

Se se dá bem com informática, você pode realizar os mesmos passos que se fazem nesta página.
Se pelo contrário não costuma usar programas de edição gráfica, esta página facilita-lhe essas tarefas.
Mesmo que você seja das pessoas a quem isto parece trivial, pense nos seus familiares, amigos, ...
Parece-lhe mais simples explicar-lhes como fazer estas operações por conta própria, ou enviar-lhes um link ou um zip para que o possam fazer de forma automatizada?.

Posso confiar que não vai fazer nada com o meu documento?

A página não armazena nenhum dado, não são usados cookies, não é usado nenhum serviço de terceiros e o seu documento só é processado no seu telemóvel/computador.

O código fonte pode vê-lo premindo F12, não está encriptado nem minimizado de forma alguma. Além disso está disponível no GitHub, pode descarregar todo o código completo.
Se quiser ter a máxima segurança, guarde a página no seu computador/telemóvel e assim sabe que o código não se vai modificar jamais.

Estou seguro após proteger assim o documento?

Não, apesar de ocultar os dados mais privados, deve sempre ter cuidado quando der o seu documento a qualquer empresa ou pessoa. Pode ler aqui as recomendações do INCIBE
É como dizer "Fechei a porta à chave ao sair de casa", isso é melhor do que deixar a porta aberta de par em par, mas um ladrão pode usar uma gazua, etc... ao fechar à chave evita os problemas mais básicos mas não é uma segurança completa.

Devo usar "Ocultar Data de Nascimento, Validade e Nº Suporte"?

Estes dados podem ser-lhe solicitados para fazer um "check-in" em alojamentos e verificar que o documento está em vigor segundo o Real Decreto 933/2021. Num dado momento Papá Friki comentou-me que por vezes o dado do Nº Suporte pode ser útil para verificar que não é um documento falsificado, e por isso também pode ser necessário deixá-lo visível.

Para que serve "Ocultar parcialmente o número do documento"?

Provavelmente não precisará de ativar esta opção. No número do documento ocultam-se os 3 primeiros e 2 últimos dígitos segundo a orientação para a aplicação provisória da disposição adicional sétima da LOPDGDD
Pode ler o pedido e caso de uso em o pedido.

Pedem-me uma cópia que não tenha marcas de água

Nesse caso basta deixar o texto em branco e o que se fará é converter para preto e branco, ocultar as partes privadas mas não se sobrescreverá nenhum texto.

Posso testar sem usar o meu documento?

Na página de testes tem as imagens de exemplo dos diversos formatos de documento. Clique em qualquer uma delas e o processo será iniciado.

Posso criar um link para esta página?

Sim, no caso de precisar pedir aos seus utilizadores que lhe enviem uma cópia do documento (certifique-se primeiro de que realmente a necessita), pode criar um link para esta página para que assim lhe enviem uma cópia já protegida e não lhe cheguem dados privados.

Além disso, nesse caso pode criar um link para a página com um parâmetro que já preencha o destinatário da cópia. São aceites os seguintes parâmetros segundo o idioma: para, for, pour, per, für (ou fur).
Exemplo: "https://password.es/id/?para=Hotel%20California" ou "https://password.es/id/en/?for=Hotel%20California".
Naturalmente o utilizador sempre poderá modificar esse texto, simplesmente pode facilitar-lhes um pouco mais o processo.

É grátis?

Sim, usar a página não tem qualquer custo.

Já uso outro sistema que gera uma marca de água que não se pode remover com IA

Lamentavelmente, isso não está realmente a proteger o seu documento, de facto pode ser que agora os seus dados estejam menos seguros do que antes já que o teatro de segurança fá-lo pensar que esse sistema está a proteger os seus dados quando não é assim.

A realidade é que um criminoso não precisa de remover a marca de água.
Se você lhe entregar uma cópia na qual estão visíveis todos os seus dados, essa pessoa abrirá um programa como o Photoshop com um modelo limpo de um documento, escreverá por cima todos os seus dados de número, nome, apelidos, data de expedição, caducidade, etc., pode querer colocar a sua própria foto e além disso ficará muito agradecido por lhe mostrar exatamente como é a sua assinatura para poder replicá-la sem problemas quando quiser assinar algo.

Portanto, siga o que diz a polícia e oculte os dados do seu documento antes de o partilhar. E certifique-se sempre de que realmente é necessário enviar-lhes o documento e que a empresa/pessoa a quem o envia é quem diz ser.
