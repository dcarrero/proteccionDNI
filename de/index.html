ID Protection

Diese Seite ermöglicht es Ihnen, die privaten Daten Ihres Ausweises einfach zu verbergen. Manchmal werden Sie nach einer Kopie Ihres Ausweises gefragt, aber Sie sollten kein Foto so senden, wie es ist, da es bei falschen Händen ernsthafte Probleme verursachen kann.

Hier können Sie unnötige Daten blockieren und einen Text überschreiben, damit sie ihn nicht für andere Zwecke verwenden können, so wird das Bild, das Sie teilen, einzigartig sein, um eine Wiederverwendung zu verhindern.

Um zu beginnen, klicken Sie auf die Schaltfläche Foto auswählen und fahren Sie mit den folgenden Schritten fort.
Dieser gesamte Prozess wird nur in Ihrem Browser durchgeführt, es werden keine Daten irgendwohin gesendet.

Fragen und Antworten

Warum sie benutzen?

Das Teilen Ihres Ausweises birgt das Risiko, dass er in falsche Hände gerät und Sie Opfer eines Identitätsdiebstahls werden.

Ist es kostenlos?

Ja, die Nutzung der Seite ist kostenlos.

Diese Seite sammelt keine Daten. Es werden keine Daten irgendwohin gesendet. Es werden keine Cookies oder Dienste von Drittanbietern verwendet.
