Ausweisschutz

Diese Seite ermoeglicht es Ihnen, einfach die privaten Daten Ihres Ausweises zu verbergen. Manchmal werden Sie gebeten, eine Kopie Ihres Ausweises zu senden, aber Sie sollten niemals einfach ein Foto senden, da es bei Missbrauch zu schwerwiegenden Problemen fuehren kann.

Hier koennen Sie die nicht benoetigten Daten blockieren und einen Text ueberlagern, damit er nicht fuer andere Zwecke verwendet werden kann. So wird das geteilte Bild einzigartig und kann nicht wiederverwendet werden.

Um zu beginnen, klicken Sie auf den Button zur Fotoauswahl und folgen Sie den naechsten Schritten.
Dieser gesamte Prozess findet nur in Ihrem Browser statt, es werden keine Daten irgendwohin gesendet.

Fragen und Antworten

Was macht diese Seite?

Es geht darum, den von der Polizei empfohlenen Schritten zum Schutz Ihres Ausweises zu folgen, ohne dass Sie sich alle merken muessen und wissen muessen, welche Daten wichtig sind.

  1. Waehlen Sie ein Foto Ihres Ausweises. Das Bild wird in Schwarzweiss umgewandelt und in der Groesse begrenzt, um keine hochwertige Kopie zu liefern.
  2. Waehlen Sie das Format Ihres Ausweises und ob es die Vorder- oder Rueckseite ist. Basierend darauf werden die nicht notwendigen Daten verborgen.
  3. Passen Sie bei Bedarf die Position des Fotos an, damit es in der Vorschau gut aussieht.
  4. Fuegen Sie einen Text hinzu, der als Wasserzeichen ueberlagert wird und angibt, wofuer Sie das Foto teilen.
  5. Klicken Sie auf den Button, um das Foto herunterzuladen.
Warum sollte ich sie nutzen?

Das Teilen Ihres Ausweises birgt das Risiko, dass er in falsche Haende geraet und Sie Opfer eines Identitaetsdiebstahls werden. Generell sollten Unternehmen keinen Zugang zu den meisten Daten Ihres Ausweises haben. Die Polizei empfiehlt, Ausweisdaten vor dem Teilen zu schuetzen.

Wenn Sie sicher sind, dass Sie Ihren Ausweis senden muessen, sollten Sie den Richtlinien der Polizei folgen, damit die Kopie schwarzweiss ist, die nicht notwendigen Daten verbirgt und einen Text ueberlagert, damit die Nutzung einzigartig ist.
Genau das macht diese Seite automatisiert.

Wenn Sie sich gut mit Computern auskennen, koennen Sie dieselben Schritte selbst durchfuehren, die diese Seite macht.
Wenn Sie hingegen normalerweise keine Grafikprogramme verwenden, erleichtert Ihnen diese Seite diese Aufgaben.
Selbst wenn Ihnen das trivial erscheint, denken Sie an Ihre Familie, Freunde...
Ist es einfacher, ihnen zu erklaeren, wie sie diese Operationen selbst durchfuehren, oder ihnen einen Link zu senden, damit sie es automatisiert machen koennen?

Kann ich darauf vertrauen, dass Sie nichts mit meinem Ausweis machen?

Die Seite speichert keine Daten, es werden keine Cookies verwendet, es werden keine Drittanbieterdienste genutzt und Ihr Ausweis wird nur auf Ihrem Handy/Computer verarbeitet.

Den Quellcode koennen Sie durch Druecken von F12 einsehen, er ist in keiner Weise verschluesselt oder minimiert. Ausserdem ist er auf GitHub verfuegbar, Sie koennen den gesamten Code herunterladen.
Wenn Sie maximale Sicherheit wuenschen, speichern Sie die Seite auf Ihrem Computer/Handy und Sie wissen, dass der Code niemals geaendert wird.

Bin ich nach dem Schutz meines Ausweises sicher?

Nein, trotz des Verbergens der privatesten Daten muessen Sie immer vorsichtig sein, wenn Sie Ihren Ausweis an ein Unternehmen oder eine Person geben. Es ist wie zu sagen "Ich habe beim Verlassen des Hauses abgeschlossen", das ist besser als die Tuer offen zu lassen, aber ein Dieb kann einen Dietrich verwenden usw... Durch das Abschliessen vermeiden Sie die grundlegendsten Probleme, aber es ist keine vollstaendige Sicherheit.

Sollte ich "Geburtsdatum, Gueltigkeit und Dokumentennummer verbergen" verwenden?

Diese Daten koennen fuer den "Check-in" in Unterkuenften angefordert werden, um zu ueberpruefen, ob der Ausweis gueltig ist. Manchmal kann die Dokumentennummer nuetzlich sein, um zu ueberpruefen, dass es sich nicht um einen gefaelschten Ausweis handelt, weshalb es auch notwendig sein kann, sie sichtbar zu lassen.

Wofuer dient "Ausweisnummer teilweise verbergen"?

Sie werden diese Option wahrscheinlich nicht aktivieren muessen. In der Ausweisnummer werden die ersten 3 und letzten 2 Ziffern gemaess Datenschutzrichtlinien verborgen.
Sie koennen die Anfrage und den Anwendungsfall in der Anfrage nachlesen.

Man verlangt eine Kopie ohne Wasserzeichen von mir

In diesem Fall lassen Sie einfach den Text leer und es wird nur in Schwarzweiss umgewandelt, die privaten Teile werden verborgen, aber es wird kein Text ueberlagert.

Kann ich testen, ohne meinen Ausweis zu verwenden?

Auf der Testseite finden Sie Beispielbilder verschiedener Ausweisformate. Klicken Sie auf eines davon und der Prozess beginnt.

Kann ich auf diese Seite verlinken?

Ja, falls Sie Ihre Benutzer bitten muessen, Ihnen eine Kopie des Ausweises zu senden (stellen Sie zuerst sicher, dass Sie sie wirklich benoetigen), koennen Sie auf diese Seite verlinken, damit sie Ihnen eine bereits geschuetzte Kopie senden und keine privaten Daten bei Ihnen ankommen.

Ausserdem koennen Sie in diesem Fall die Seite mit einem Parameter verlinken, der den Empfaenger der Kopie bereits ausfuellt. Folgende Parameter werden je nach Sprache akzeptiert: para, for, pour, per, fuer (oder fur).
Beispiel: "https://password.es/id/de/?fuer=Hotel%20California".
Natuerlich kann der Benutzer diesen Text immer aendern, es erleichtert ihm nur ein wenig den Prozess.

Ist es kostenlos?

Ja, die Nutzung der Seite ist voellig kostenlos.

Ich verwende bereits ein anderes System, das ein Wasserzeichen erzeugt, das nicht mit KI entfernt werden kann

Leider schuetzt das Ihren Ausweis nicht wirklich, tatsaechlich koennten Ihre Daten jetzt weniger sicher sein als zuvor, da das Sicherheitstheater Sie glauben laesst, dass dieses System Ihre Daten schuetzt, obwohl es das nicht tut.

Die Realitaet ist, dass ein Krimineller das Wasserzeichen nicht entfernen muss.
Wenn Sie ihm eine Kopie geben, auf der alle Ihre Daten sichtbar sind, wird diese Person ein Programm wie Photoshop mit einer sauberen Ausweisvorlage oeffnen, alle Ihre Daten wie Ausweisnummer, Name, Nachname, Ausstellungsdatum, Ablaufdatum usw. darueber schreiben, vielleicht sein eigenes Foto einsetzen und ausserdem sehr dankbar sein, dass Sie ihm genau zeigen, wie Ihre Unterschrift aussieht, damit er sie problemlos replizieren kann, wenn er etwas unterschreiben moechte.

Hoeren Sie daher auf die Polizei und verbergen Sie die Daten Ihres Ausweises, bevor Sie ihn teilen. Und stellen Sie immer sicher, dass es wirklich notwendig ist, den Ausweis zu senden und dass das Unternehmen/die Person, an die Sie ihn senden, wirklich die ist, die sie vorgibt zu sein.
