Ausweisschutz

Diese Seite ermöglicht es Ihnen, einfach die privaten Daten Ihres Ausweises zu verbergen. Manchmal werden Sie gebeten, eine Kopie Ihres Ausweises zu senden, aber Sie sollten niemals einfach ein Foto senden, da es bei Missbrauch zu schwerwiegenden Problemen führen kann.

Hier können Sie die nicht benötigten Daten blockieren und einen Text überlagern, damit er nicht für andere Zwecke verwendet werden kann. So wird das geteilte Bild einzigartig und kann nicht wiederverwendet werden.

Um zu beginnen, klicken Sie auf den Button zur Fotoauswahl und folgen Sie den nächsten Schritten.
Dieser gesamte Prozess findet nur in Ihrem Browser statt, es werden keine Daten irgendwohin gesendet.

Fragen und Antworten

Was macht diese Seite?

Es geht darum, den von der Polizei empfohlenen Schritten zum Schutz Ihres Ausweises zu folgen, ohne dass Sie sich alle merken müssen und wissen müssen, welche Daten wichtig sind.

  1. Wählen Sie ein Foto Ihres Ausweises. Das Bild wird in Schwarzweiß umgewandelt und in der Größe begrenzt, um keine hochwertige Kopie zu liefern.
  2. Wählen Sie das Format Ihres Ausweises und ob es die Vorder- oder Rückseite ist. Basierend darauf werden die nicht notwendigen Daten verborgen.
  3. Passen Sie bei Bedarf die Position des Fotos an, damit es in der Vorschau gut aussieht.
  4. Fügen Sie einen Text hinzu, der als Wasserzeichen überlagert wird und angibt, wofür Sie das Foto teilen.
  5. Klicken Sie auf den Button, um das Foto herunterzuladen.
Warum sollte ich sie nutzen?

Das Teilen Ihres Ausweises birgt das Risiko, dass er in falsche Hände gerät und Sie Opfer eines Identitätsdiebstahls werden. Generell sollten Unternehmen keinen Zugang zu den meisten Daten Ihres Ausweises haben. Die Polizei empfiehlt, Ausweisdaten vor dem Teilen zu schützen.

Wenn Sie sicher sind, dass Sie Ihren Ausweis senden müssen, sollten Sie den Richtlinien der Polizei folgen, damit die Kopie schwarzweiß ist, die nicht notwendigen Daten verbirgt und einen Text überlagert, damit die Nutzung einzigartig ist.
Genau das macht diese Seite automatisiert.

Wenn Sie sich gut mit Computern auskennen, können Sie dieselben Schritte selbst durchführen, die diese Seite macht.
Wenn Sie hingegen normalerweise keine Grafikprogramme verwenden, erleichtert Ihnen diese Seite diese Aufgaben.
Selbst wenn Ihnen das trivial erscheint, denken Sie an Ihre Familie, Freunde...
Ist es einfacher, ihnen zu erklären, wie sie diese Operationen selbst durchführen, oder ihnen einen Link zu senden, damit sie es automatisiert machen können?

Kann ich darauf vertrauen, dass Sie nichts mit meinem Ausweis machen?

Die Seite speichert keine Daten, es werden keine Cookies verwendet, es werden keine Drittanbieterdienste genutzt und Ihr Ausweis wird nur auf Ihrem Handy/Computer verarbeitet.

Den Quellcode können Sie durch Drücken von F12 einsehen, er ist in keiner Weise verschlüsselt oder minimiert. Außerdem ist er auf GitHub verfügbar, Sie können den gesamten Code herunterladen.
Wenn Sie maximale Sicherheit wünschen, speichern Sie die Seite auf Ihrem Computer/Handy und Sie wissen, dass der Code niemals geändert wird.

Bin ich nach dem Schutz meines Ausweises sicher?

Nein, trotz des Verbergens der privatesten Daten müssen Sie immer vorsichtig sein, wenn Sie Ihren Ausweis an ein Unternehmen oder eine Person geben. Es ist wie zu sagen "Ich habe beim Verlassen des Hauses abgeschlossen", das ist besser als die Tür offen zu lassen, aber ein Dieb kann einen Dietrich verwenden usw... Durch das Abschließen vermeiden Sie die grundlegendsten Probleme, aber es ist keine vollständige Sicherheit.

Sollte ich "Geburtsdatum, Gültigkeit und Dokumentennummer verbergen" verwenden?

Diese Daten können für den "Check-in" in Unterkünften angefordert werden, um zu überprüfen, ob der Ausweis gültig ist. Manchmal kann die Dokumentennummer nützlich sein, um zu überprüfen, dass es sich nicht um einen gefälschten Ausweis handelt, weshalb es auch notwendig sein kann, sie sichtbar zu lassen.

Wofür dient "Ausweisnummer teilweise verbergen"?

Sie werden diese Option wahrscheinlich nicht aktivieren müssen. In der Ausweisnummer werden die ersten 3 und letzten 2 Ziffern gemäß Datenschutzrichtlinien verborgen.
Sie können die Anfrage und den Anwendungsfall in der Anfrage nachlesen.

Man verlangt eine Kopie ohne Wasserzeichen von mir

In diesem Fall lassen Sie einfach den Text leer und es wird nur in Schwarzweiß umgewandelt, die privaten Teile werden verborgen, aber es wird kein Text überlagert.

Kann ich testen, ohne meinen Ausweis zu verwenden?

Auf der Testseite finden Sie Beispielbilder verschiedener Ausweisformate. Klicken Sie auf eines davon und der Prozess beginnt.

Kann ich auf diese Seite verlinken?

Ja, falls Sie Ihre Benutzer bitten müssen, Ihnen eine Kopie des Ausweises zu senden (stellen Sie zuerst sicher, dass Sie sie wirklich benötigen), können Sie auf diese Seite verlinken, damit sie Ihnen eine bereits geschützte Kopie senden und keine privaten Daten bei Ihnen ankommen.

Außerdem können Sie in diesem Fall die Seite mit einem Parameter verlinken, der den Empfänger der Kopie bereits ausfüllt. Folgende Parameter werden je nach Sprache akzeptiert: para, for, pour, per, für (oder fur).
Beispiel: "https://password.es/id/de/?für=Hotel%20California".
Natürlich kann der Benutzer diesen Text immer ändern, es erleichtert ihm nur ein wenig den Prozess.

Ist es kostenlos?

Ja, die Nutzung der Seite ist völlig kostenlos.

Ich verwende bereits ein anderes System, das ein Wasserzeichen erzeugt, das nicht mit KI entfernt werden kann

Leider schützt das Ihren Ausweis nicht wirklich, tatsächlich könnten Ihre Daten jetzt weniger sicher sein als zuvor, da das Sicherheitstheater Sie glauben lässt, dass dieses System Ihre Daten schützt, obwohl es das nicht tut.

Die Realität ist, dass ein Krimineller das Wasserzeichen nicht entfernen muss.
Wenn Sie ihm eine Kopie geben, auf der alle Ihre Daten sichtbar sind, wird diese Person ein Programm wie Photoshop mit einer sauberen Ausweisvorlage öffnen, alle Ihre Daten wie Ausweisnummer, Name, Nachname, Ausstellungsdatum, Ablaufdatum usw. darüber schreiben, vielleicht sein eigenes Foto einsetzen und außerdem sehr dankbar sein, dass Sie ihm genau zeigen, wie Ihre Unterschrift aussieht, damit er sie problemlos replizieren kann, wenn er etwas unterschreiben möchte.

Hören Sie daher auf die Polizei und verbergen Sie die Daten Ihres Ausweises, bevor Sie ihn teilen. Und stellen Sie immer sicher, dass es wirklich notwendig ist, den Ausweis zu senden und dass das Unternehmen/die Person, an die Sie ihn senden, wirklich die ist, die sie vorgibt zu sein.
